CIPHER TIME COMPARISON USING CLASSICAL AND QUANTUM APPROACHES

This project successfully explored the implementation and performance analysis of classical cryptographic algorithms—such as Playfair, Hill, and DES—within a quantum computing environment using Qiskit. By implementing these ciphers in both classical and quantum platforms, we gained insights into their computational behaviour, execution time, and scalability.

The study revealed that while quantum can replicate classical encryption logic, they generally exhibit higher execution time due to overhead and complexity in circuit design. This highlights the current limitations of quantum, particularly when run on classical machines.

🚀 Just Built My Own Chatbot Using Microsoft Power Virtual Agents! 🤖✨

Excited to share a recent project where I created a smart chatbot powered by Microsoft Power Virtual Agents! I integrated an Excel knowledge base containing customer order details, enabling the bot to intelligently respond to queries about their orders.
